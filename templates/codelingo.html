Essential Programming Languages

  • C: A foundational procedural language known for speed and low-level memory access. Used in system programming, embedded systems, and operating systems.
  • C++: An extension of C with object-oriented features. Widely used in game development, high-performance applications, and competitive programming.
  • Python: A high-level, readable language popular for data science, automation, web development, and AI/ML. Known for its vast ecosystem and beginner-friendly syntax.
  • Java: A class-based, object-oriented language used in enterprise applications, Android development, and backend systems. Runs on the JVM (Java Virtual Machine).
  • Go (Golang): Developed by Google, known for simplicity, concurrency support, and performance. Ideal for cloud services, APIs, and scalable systems.
  • JavaScript: The language of the web. Powers dynamic frontend behavior and, with Node.js, backend services too. Core to modern web development.
  • SQL: Structured Query Language used to interact with relational databases. Essential for data retrieval, manipulation, and reporting.
  • Bash: A Unix shell scripting language used for automating tasks, managing servers, and writing command-line utilities.

System-Defined Functions in C

  • stdio.h – Standard Input/Output:
    • printf() – Prints formatted output to the console
    • scanf() – Reads formatted input from the user
    • gets(), puts() – Read/write strings (deprecated: use fgets())
    • fopen(), fclose() – Open/close files
    • fread(), fwrite() – Read/write binary data
    • fprintf(), fscanf() – File-based formatted I/O
  • stdlib.h – General Utilities:
    • malloc(), calloc(), realloc() – Dynamic memory allocation
    • free() – Deallocates memory
    • exit() – Terminates the program
    • atoi(), atof(), strtol() – Convert strings to numbers
    • rand(), srand() – Generate random numbers
    • system() – Executes shell commands
  • string.h – String Manipulation:
    • strlen() – Returns length of a string
    • strcpy(), strncpy() – Copy strings
    • strcat(), strncat() – Concatenate strings
    • strcmp(), strncmp() – Compare strings
    • strchr(), strstr() – Search within strings
    • memset(), memcpy() – Memory operations
  • math.h – Mathematical Functions:
    • pow(), sqrt() – Power and square root
    • sin(), cos(), tan() – Trigonometric functions
    • log(), exp() – Logarithmic and exponential
    • ceil(), floor(), fabs() – Rounding and absolute value
  • time.h – Time and Date:
    • time() – Returns current time
    • clock() – Processor time used
    • difftime() – Difference between two times
    • strftime() – Format time as string
  • ctype.h – Character Handling:
    • isalpha(), isdigit(), isalnum() – Character checks
    • toupper(), tolower() – Case conversion

System-Defined Functions in Java

  • System Class (java.lang.System):
    • System.out.println() – Print output to console
    • System.in.read() – Read input from console
    • System.currentTimeMillis() – Get current time in milliseconds
    • System.exit() – Terminate the program
  • Math Class (java.lang.Math):
    • Math.pow(), Math.sqrt() – Power and square root
    • Math.abs(), Math.max(), Math.min() – Absolute and comparisons
    • Math.sin(), Math.cos(), Math.tan() – Trigonometric functions
    • Math.random() – Generate random number between 0.0 and 1.0
  • String Class (java.lang.String):
    • length() – Returns string length
    • charAt() – Access character at index
    • substring() – Extract substring
    • indexOf(), lastIndexOf() – Search within string
    • toUpperCase(), toLowerCase() – Case conversion
    • equals(), equalsIgnoreCase() – Compare strings
    • trim(), replace(), split() – Modify strings
  • Arrays Class (java.util.Arrays):
    • sort() – Sort array
    • copyOf(), copyOfRange() – Copy arrays
    • fill() – Fill array with value
    • equals() – Compare arrays
    • toString() – Convert array to string
  • Collections Class (java.util.Collections):
    • sort() – Sort list
    • reverse() – Reverse list
    • shuffle() – Randomize list order
    • max(), min() – Find max/min in collection
  • Scanner Class (java.util.Scanner):
    • nextLine(), nextInt(), nextDouble() – Read user input
    • hasNext() – Check for more input
    • close() – Close scanner
  • File Class (java.io.File):
    • exists(), createNewFile() – Check/create file
    • delete() – Delete file
    • length() – Get file size
    • getName(), getPath() – File info
  • Thread Class (java.lang.Thread):
    • start() – Begin thread execution
    • sleep() – Pause thread
    • join() – Wait for thread to finish
    • getName(), setName() – Manage thread identity
